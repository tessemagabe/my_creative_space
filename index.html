About Me

I am a results-driven Software Test Automation Engineer passionate about designing efficient, scalable, and reliable testing frameworks that uphold the highest standards of software quality. My expertise lies in crafting automation solutions that enhance test coverage, accelerate development cycles, and integrate seamlessly into CI/CD pipelines.

Holding advanced degrees in Information Systems and Business Administration, along with certifications in DevOps Engineering and Cybersecurity Analysis, I bridge the gap between technical innovation and strategic business objectives. This unique blend allows me to develop test automation strategies that align with industry best practices and company goals while maintaining agility in today’s fast-paced development environments.

Featured Project: BDD Cucumber JUnit Test Automation

I recently completed a BDD Cucumber JUnit project, which is now available on my GitHub repository. This project showcases my ability to implement Behavior-Driven Development (BDD) principles using Cucumber with JUnit, fostering clear communication between technical and non-technical stakeholders while improving software testability.

Key Highlights:

  • Designed structured, reusable test scenarios using Gherkin syntax for better readability and maintainability.
  • Integrated with JUnit, ensuring efficient test execution and reporting.

  • Included detailed documentation to guide users on setup, execution, and best practices.
